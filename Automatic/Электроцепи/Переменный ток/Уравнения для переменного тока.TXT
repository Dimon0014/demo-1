Эти примеры иллюстрируют использование блоков решения
распределённой СЛАУ для задач электротехники
в приближении многочастотной схемы синусоидального 
переменного тока.
Данная методика позволяет считать электроэнергетические системы
переменного тока с наличием разночастотных источников.

Основные используемые соотношения:

Комплексная проводимость ёмкости:

Xc = 0 + j*(2*pi*f)*C   

Комплексная проводимость индуктивности:

XL = 0 - j*(1/(2*pi*f*L))

Уравнения баланса токов узла для переменного тока в комплексных числах:

u1 ----- u ----- u2

k1 - проводимость от u1 к u
k2 - проводимость от u к u2

ure*(k2im + k1im) + uim*(k1re + k2re) - u1re*k1im - u1im*k1re - u2re*k2im - u2im*k2re = 0
ure*(k1re + k2re) - uim*(k1im + k2im) - u1re*k1re + u1im*k1im - u2re*k2re + u2im*k2im = 0

Вычисления активного и реаквтивного тока:

ire = ure*kre - uim*kim;
iim = ure*kim + uim*kre;
