Пример работает только в 32-битной версии SimInTech.

Для работы примера необходимо:
1) Открыть пример проект "Источник.prt" и "Приемник.prt".
2) Запустить проект "Источник.prt", после чего запустить проект "Приемник.prt". 

В менеджере данных обоих проектом имеется элемент "Экспорт/импорт в общую память".
