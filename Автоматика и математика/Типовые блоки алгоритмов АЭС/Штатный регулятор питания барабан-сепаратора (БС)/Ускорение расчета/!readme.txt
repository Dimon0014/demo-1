В проектах "тест_скорости_DLL.prt" и "тест_скорости_схема.prt" происходит расчет одного и того же алгоритма, выполненного в виде расчетной схемы и динамической библиотеки. На данном примере можно видеть насколько сильно увеличивается скорость счета, с использованием генерации DLL.

Папка "блоки регулятора" содержит необходимые субмодели для запуска проекта "тест_скорости_схема.prt".
