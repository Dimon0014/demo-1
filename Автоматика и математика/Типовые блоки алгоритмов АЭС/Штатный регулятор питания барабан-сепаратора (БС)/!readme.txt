Представлена модель штатного регулятора питания барабан-сепаратора атомной электростанции. Запускать следует файл "ШРП - штатный регулятор питания БС.prt"
В папке "Ускорение расчета" доступен пример ускорения расчета данной схемы за счет использованнания автоматически сгенерированной DLL-библиотеки.

Папка "блоки регулятора" содержит необходимые субмодели для запуска проекта "тест_скорости_схема.prt".
